## Quick Example

**Zero-Config Setup:**
```python
from zenith import Zenith

# 🎯 Zero-config - just works!
app = Zenith()

@app.get("/")
async def hello():
    return {"message": "Modern Python web development!"}
```

**Enhanced Models:**
```python
from zenith import Zenith
from zenith.core import DB
from zenith.db import ZenithModel
from sqlmodel import Field
from typing import Optional

app = Zenith()

# 🏗️ Enhanced model with intuitive patterns
class User(ZenithModel, table=True):
    id: Optional[int] = Field(primary_key=True)
    name: str = Field(max_length=100)
    email: str = Field(unique=True)
    active: bool = Field(default=True)

@app.get("/users")
async def list_users():  # ✨ No session management!
    # Chainable queries work automatically
    query = await User.where(active=True)
    users = await query.order_by('-id').limit(10).all()
    return {"users": [user.to_dict() for user in users]}

@app.post("/users")
async def create_user(user_data: dict):
    # Simple: User.create() - no session management!
    user = await User.create(**user_data)
    return {"user": user.to_dict()}
```

**One-liner Features:**
```python
from zenith import Zenith

app = Zenith()

# ⚡ Add features in one line each
app.add_auth()    # JWT authentication + /auth/login
app.add_admin()   # Admin dashboard at /admin
app.add_api("My API", "1.0.0")  # API docs at /docs

# 🌐 Serve your SPA
app.spa("dist")
```

## Key Features

### Maximum Productivity

- 🚀 **85% Less Boilerplate** - Zero-config setup with intelligent defaults
- 🏗️ **Intuitive Patterns** - `User.where().order_by().limit().all()` database operations
- ⚡ **One-liner Features** - `app.add_auth()`, `app.add_admin()`, `app.add_api()`
- 🎯 **Enhanced DX** - ZenithModel handles sessions automatically
- 🔍 **Automatic 404s** - `User.find_or_404()` with proper error handling
- 📦 **Request-scoped Sessions** - No manual database session management

### Production Ready

- ✅ **631+ Tests Passing** - Comprehensive test coverage with zero memory leaks
- 🏎️ **9,600+ req/s** - Exceptional performance with all features enabled
- 🛡️ **Security by Default** - CORS, CSRF, security headers automatic
- 🏥 **Built-in Monitoring** - `/health`, `/metrics`, request tracing
- ⚡ **Background Tasks** - TaskGroups with Python 3.12+ optimizations
- 🌐 **SPA Support** - One command: `app.spa("dist")`

## Philosophy

Zenith follows these core principles:

1. **Maximum Productivity** - Zero-config setup and intuitive patterns eliminate boilerplate
2. **Type Safety** - Full type hints and Pydantic validation catch errors at development time
3. **Performance First** - 9,600+ req/s with no performance penalty for convenience features
4. **Developer Joy** - One-liner features and enhanced DX maximize Python productivity
5. **Production Focus** - Security, monitoring, and scalability built-in from day one
